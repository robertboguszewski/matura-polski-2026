🚀 3 sposoby użycia tego repo

WEB

Przeglądarka (GitHub Pages)

Otwierasz tę stronę, ustawiasz klucz API Anthropic i klikasz dowolne narzędzie. Wszystko działa client-side, tylko wywołania do api.anthropic.com idą bezpośrednio z Twojej przeglądarki.
COWORK

Claude Cowork (desktop)

Sklonuj repo, otwórz folder w Cowork. Artifacty automatycznie używają window.cowork.sample() (Haiku, billing z Twojego planu).
CODE

Claude Code (CLI)

Sklonuj repo, otwórz w VS Code z Claude Code. Wbudowany agent korzysta z vault (folder vault/) jako bazy wiedzy. Czytaj CLAUDE.md dla instrukcji rola korepetytora.
