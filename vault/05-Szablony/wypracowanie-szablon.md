# 📝 Szablon rozprawki maturalnej (poziom podstawowy, 35 pkt)

> **Czas pisania:** 180 min | **Minimum słów:** 300 | **Struktura:** 5 akapitów (teza → 3 argumenty → konkluzja)
> **Kryteria CKE 2023:** 6 + 4 + 16 + 2 + 4 + 2 + 1 = 35 pkt

---

## 🎯 PRZED PISANIEM (15 min — na brudnopisie)

### Krok 1: Rozbij temat (2 min)
- **Co jest problemem?** (podkreśl kluczowe słowo w temacie)
- **Jakie jest pytanie?** (przeformułuj temat jako pytanie, na które odpowiadasz)
- **Czy temat wymaga tak/nie czy "zależy od…"?**

Wpisz temat: `_______________________________________________`

### Krok 2: Postaw tezę (3 min)
Teza = **jedno zdanie, które jest Twoją odpowiedzią** na pytanie tematu.

**Schemat tezy:** *"[X] wpływa na życie człowieka, ponieważ [Y] — potwierdzają to: [lektura 1], [lektura 2], [lektura 3]."*

Twoja teza: `_______________________________________________`

### Krok 3: Wybierz 3 lektury (5 min)
**Zasada:** 1 lektura obowiązkowa CAŁOŚCIOWA (dłuższy argument) + 2 konteksty (krótsze).

- **Argument 1 (lektura obowiązkowa całościowa):** `_______` → scena/bohater: `_______`
- **Argument 2 (kontekst krótszy):** `_______` → fragment: `_______`
- **Argument 3 (kontekst 3):** `_______` → motyw: `_______`

### Krok 4: Plan akapitów (5 min)
| Akapit | Co? | Ile słów |
|---|---|---|
| 1. Wstęp + teza | kontekst + przedstaw tezę | 40–60 |
| 2. Argument 1 (główna lektura) | PEEL: Point → Evidence → Explain → Link | 80–110 |
| 3. Argument 2 (kontekst) | PEEL | 60–80 |
| 4. Argument 3 (kontekst) | PEEL | 60–80 |
| 5. Konkluzja | powtórz tezę innym językiem + refleksja | 40–60 |
| **RAZEM** | | **~300-400** |

---

## 📖 AKAPIT 1 — WSTĘP + TEZA (40-60 słów)

**Schemat:**
1. Zdanie ogólne (kulturowe/filozoficzne) — "Od dawna ludzie zadają sobie pytanie…"
2. Nawiązanie do tematu — "Warto zastanowić się, czy…"
3. **TEZA** (wyraźna!) — "Uważam, że [X], co potwierdzają utwory [A], [B] i [C]."

**Zwroty użyteczne:**
- "Problem [X] jest uniwersalny — pojawia się w literaturze wszystkich epok…"
- "Warto zastanowić się, czy…"
- "Moim zdaniem…"
- "Literatura daje nam bogaty materiał, by odpowiedzieć…"

---

## 📖 AKAPIT 2 — ARGUMENT 1 (lektura obowiązkowa, 80-110 słów)

**Schemat PEEL:**
- **P (Point):** zdanie tematyczne — co ten akapit udowadnia
- **E (Evidence):** konkretna scena/cytat z lektury
- **E (Explain):** jak to potwierdza tezę
- **L (Link):** płynne przejście do następnego akapitu

**Zwroty użyteczne:**
- "Pierwszym argumentem jest…" / "Warto zacząć od…"
- "W [lektura] obserwujemy, jak [bohater] [akcja]…"
- "Przykładem tego jest scena, w której…"
- "Świadczy to o tym, że…"
- "Podobną postawę odnajdujemy w…" *(link do akapitu 3)*

**Uwaga:** TO JEST NAJWAŻNIEJSZY AKAPIT. Tu CKE sprawdza "kompetencje literackie" (16 pkt z 35).

---

## 📖 AKAPIT 3 — ARGUMENT 2 (kontekst, 60-80 słów)

**Struktura jak wyżej (PEEL), ale krócej.**

**Zwroty użyteczne:**
- "Kolejnym przykładem jest…"
- "Warto również zwrócić uwagę na…"
- "Inaczej przedstawia ten problem…"
- "Dzieło [X] pokazuje, że…"

---

## 📖 AKAPIT 4 — ARGUMENT 3 (kontekst, 60-80 słów)

**Schemat PEEL + link do konkluzji.**

**Zwroty użyteczne:**
- "Ostatnim przykładem niech będzie…"
- "Nie można pominąć…"
- "W tym kontekście warto też wspomnieć o…"

---

## 📖 AKAPIT 5 — KONKLUZJA (40-60 słów)

**Schemat:**
1. Powtórz tezę **innymi słowami** (nie 1:1 ze wstępu)
2. Krótkie podsumowanie 3 argumentów
3. Uniwersalne zdanie-refleksja (nie moralizatorskie!)

**Zwroty użyteczne:**
- "Podsumowując, można stwierdzić, że…"
- "Omówione przykłady pokazują, że…"
- "Literatura od wieków uczy nas, że…"
- "Być może właśnie dlatego literatura pozostaje…"

---

## ✅ CHECKLIST PRZED ODDANIEM (5 min)

### Treść
- [ ] Teza jest **jasna** i odpowiada na pytanie z tematu
- [ ] Każdy akapit ma **zdanie tematyczne** (Point)
- [ ] Użyłem **dokładnego cytatu lub sceny** (nie ogólnika typu "Wokulski walczył o miłość")
- [ ] Min. 1 lektura **obowiązkowa całościowa** (nie tylko fragment)
- [ ] **3 różne utwory** — nie ten sam 3 razy
- [ ] Konkluzja powtarza tezę **innym językiem**

### Styl
- [ ] Brak powtórzeń (sprawdź "bohater", "autor", "można powiedzieć")
- [ ] Brak kolokwializmów ("fajny", "super", "beka")
- [ ] Zdania **urozmaicone** długością (nie same krótkie / długie)
- [ ] Słownictwo analityczne: "podmiot mówiący", "narrator", "protagonist"

### Język
- [ ] Interpunkcja (przecinki przed "który", "że", "ponieważ")
- [ ] Zgodność podmiotu z orzeczeniem
- [ ] Brak błędów gramatycznych (sprawdź: "sama siebie" vs "samego siebie")

### Ortografia (1 pkt)
- [ ] Imiona bohaterów pisane wielką literą
- [ ] "nie" z czasownikami osobno
- [ ] Tytuły lektur kursywą lub w cudzysłowie

---

## ❌ CZEGO UNIKAĆ

- **Streszczanie fabuły zamiast argumentowania** — to klasyczny błąd, maksymalnie 2-3 zdania fabuły, reszta = analiza
- **"Autor chciał pokazać…"** — nigdy nie wiesz, co autor chciał. Używaj: "W utworze ukazany jest…", "Tekst pokazuje, że…"
- **Cytaty bez omówienia** — każdy cytat = 1-2 zdania wyjaśnienia
- **Osobiste anegdoty** ("Mój dziadek mówił…") — matura to nie blog
- **Zbyt szerokie tezy** ("Życie jest trudne") — bądź konkretny
- **Wnioski moralizatorskie** ("Każdy powinien kochać ojczyznę") — refleksja, nie kazanie
