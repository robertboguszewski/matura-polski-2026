## 🏛 ANTYK

| Pytanie | Odpowiedź |
|---------|-----------|
| Główne wartości? | Prawda, dobro, piękno (triada platońska). Harmonia, umiar. |
| **Mimesis** — co to? | Sztuka naśladuje rzeczywistość (Platon, Arystoteles). |
| **Katharsis** — co to? | Oczyszczenie widza poprzez litość i trwogę (*Poetyka* Arystotelesa). |
| Gatunki kluczowe? | Epos (*Iliada*), tragedia (*Antygona*), oda, hymn, elegia. |
| Autorzy? | Homer, Sofokles, Arystoteles, Wergiliusz, Horacy. |
| Motywy biblijne? | Hiob, marność, syn marnotrawny, dekalog — wracają w całej lit. polskiej. |

## 💀 BAROK

| Pytanie | Odpowiedź |
|---------|-----------|
| Nastrój? | Niepokój metafizyczny, dualizm, kontrrreformacja. |
| **Vanitas** — źródło? | "Vanitas vanitatum" (Księga Koheleta). Sęp-Szarzyński, Naborowski. |
| **Marinizm** vs **sarmatyzm**? | Marinizm = wyrafinowana poezja dworska (Morsztyn). Sarmatyzm = ziemiańskie PL (Pasek). |
| Gatunki? | Sonet, emblemat, pamiętnik, epos (*Wojna chocimska* Potockiego). |
| Hak pamięciowy? | BARK = **B**ohaterowie **A**mbiwalentni, **R**ozdarcie, **K**ontrreformacja. |
